# DocuShield Frontend

This is a lightweight React prototype for the DocuShield project.

## Install

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

The application will start on `http://localhost:5173` and proxy API calls to the backend.

## One-Command Startup (Windows)

From the repository root you can also run:

```powershell
.\run.ps1
```
